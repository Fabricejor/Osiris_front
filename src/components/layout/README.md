# Layout Components

Ce dossier (`src/components/layout`) contient les composants structurels qui définissent l'agencement global des pages ou des grandes sections de l'application.

## Règles
- Ces composants servent de conteneurs ou de coquilles pour d'autres contenus.
- Exemples : `Sidebar`, `Header`, `Footer`, `PageContainer`, `DashboardLayout`, etc.
- Ils gèrent souvent le comportement "responsive" de la structure globale de l'application (ex: menu hamburger sur mobile, barre latérale sur desktop).
